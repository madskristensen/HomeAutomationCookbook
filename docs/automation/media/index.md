## Best Practices

### ‚úÖ Speaker Placement

**Optimal Audio:**
- Kitchen: Above cabinets or on counter
- Bathroom: Away from direct water (IP rating)
- Bedroom: Nightstand or dresser
- Living room: Multiple speakers for stereo
- Outdoor: Weather-resistant models

### üì° Network Considerations

**Reliability:**
- Use 2.4GHz WiFi (better range)
- Wired ethernet when possible
- Keep speakers away from microwave
- Update firmware regularly
- Dedicated WiFi network for smart home (optional)

### üéöÔ∏è Volume Strategy

**Time-Based Volumes:**
- **Morning (6-10 AM):** Low 20-30%
- **Daytime (10 AM-6 PM):** Medium 40-50%
- **Evening (6-10 PM):** Medium 40-50%
- **Late night (10 PM+):** Low 15-25%

**Room-Specific:**
- Bedroom: Always quieter
- Kitchen: Can be louder (appliance noise)
- Living room: Flexible based on activity
- Bathroom: Medium (echoes amplify)

### üé¨ Media Automation

**TV Integration Methods:**

1. **Power Monitoring:**
   - Smart plug on TV
   - Detects power draw
   - Simple and reliable

2. **Smart TV Integration:**
   - Chromecast, Fire TV, Roku
   - Detect specific apps
   - More advanced control

3. **HDMI-CEC:**
   - TV communicates with other devices
   - Requires compatible hardware
   - Most reliable for state detection

### üîÑ Daily Reset

**Why Reset at 4 AM:**
- Before anyone wakes up
- Ensures consistent starting point
- Fixes volume drift from manual changes
- Dissolves temporary speaker groups
- Ready for morning routines

**What to Reset:**
- Volume levels (speakers and assistants)
- Speaker groups (back to individual)
- Playback states (stop any playing music)
- Audio outputs (restore defaults)

## Troubleshooting

**Music Not Starting Automatically?**
- Verify trigger (light, door, presence)
- Check music service is linked
- Ensure speaker is powered on
- Test speaker responds to manual play
- Check automation is enabled

**Lights Not Dimming for TV?**
- Verify TV power monitoring detects on state
- Check threshold is correct (usually 50W+)
- Test lights respond to manual commands
- Ensure automation time conditions are met

**Volume Too Loud/Quiet?**
- Check last manual volume change
- Verify daily reset automation runs
- Adjust preset volume levels
- Check speaker-specific settings

**Speaker Groups Not Working?**
- Ensure all speakers on same network
- Check for firmware updates
- Verify platform supports grouping
- Test creating group manually first

**Shower Music Stops Too Soon?**
- Increase timer duration
- Use "door closed" instead of "lights off"
- Add delay before stopping
- Check for conflicting automations
