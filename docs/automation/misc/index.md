## Common patterns

### Power monitoring detection

Many misc automations use power monitoring to detect device usage:

**Setup:**
1. Plug device into power monitoring smart plug
2. Observe normal power consumption
3. Create automation when power rises/falls
4. Use as trigger or condition

**Examples:**
- White noise machine: Baby sleeping when > 5W
- Electric toothbrush charger: Used when > 2W
- Doorbell transformer: Ring when power spikes

### Time-based schedules

Many misc automations are time-driven:

**Patterns:**
- Sunset/sunrise triggers
- Specific time triggers
- Day of week conditions
- Time range conditions

**Examples:**
- Outdoor lights at sunset
- Garbage reminder Wednesday 6 PM
- Teeth reminder 8-10 PM window

### Virtual switches for state

Track state across automations:

**Pattern:**
- Create virtual switch or input boolean
- Toggle on/off based on events
- Use as condition in other automations

**Examples:**
- "Baby Sleeping" switch
- "Garbage Out" switch
- "Dishwasher Clean" switch

## Best practices

### Keep it simple

**Good Misc Automations:**
‚úÖ Single purpose
‚úÖ Easy to understand
‚úÖ Predictable behavior
‚úÖ Clear benefit

**Avoid:**
‚ùå Complex multi-step logic

### Family acceptance

**Make It Helpful:**
- Solve real problems
- Don't be annoying
- Easy to disable
- Explain what it does

**Get Buy-In:**
- Involve family in planning
- Address concerns
- Iterate based on feedback
- Provide manual overrides

### Low maintenance

**Design for Reliability:**
- Use reliable triggers
- Account for edge cases
- Test thoroughly
- Document what you built

**Minimize Dependencies:**
- Fewer devices = fewer failure points
- Local processing when possible
- Simple logic preferred

## Additional automation ideas

### Mailbox notification

Know when mail arrives:

**Setup:**
- Contact sensor in mailbox
- Trigger when opened
- Send notification

**Considerations:**
- Weatherproof sensor needed
- Battery life in outdoor conditions
- May trigger on outgoing mail too

### Pet door tracking

Know when pets go out:

**Setup:**
- Contact sensor on pet door
- Track open/close events
- Log or notify

**Extensions:**
- Track how long outside
- Notify if door stuck open
- Combine with outdoor camera

### Medication reminders

Don't forget medications:

**Setup:**
- Contact sensor on medicine cabinet
- If not opened by certain time, remind
- Track compliance

**Logic:**
- Trigger: Time is 8 AM
- Condition: Cabinet not opened today
- Action: Send reminder

### School morning routine

Help get kids out on time:

**Setup:**
- Time-based triggers
- Announcements on smart speakers
- Progressive reminders

**Example:**
- 6:30 AM: "Good morning, time to wake up!"
- 7:00 AM: "Breakfast time!"
- 7:30 AM: "Get dressed and ready!"
- 7:50 AM: "Bus comes in 10 minutes!"

### Delivery detection

Know when packages arrive:

**Setup:**
- Motion sensor on porch
- Camera person detection
- Notification when triggered

**Extensions:**
- Dashboard tile until brought in
- Different alerts for expected vs. unexpected
- Time-based (delivery hours only)
