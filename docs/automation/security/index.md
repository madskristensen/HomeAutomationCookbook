# Security and safety automations

A lot of people start their smart home because they want added security. That's typically done by adding camera systems both indoors and outdoors around the house. However, there are plenty of other automations we can add to help secure the house or make us feel safer.

## Troubleshooting

<div class="troubleshooting-grid">

<div class="issue-card">
<div class="issue-header">
<h3>False alarms</h3>
</div>
<div class="issue-solutions">
<ul>
<li>Use time delays before major actions</li>
<li>Require multiple triggers (smoke + motion)</li>
<li>Exclude kitchen from immediate response</li>
<li>Use pet-immune motion sensors</li>
</ul>
</div>
</div>

<div class="issue-card">
<div class="issue-header">
<h3>Notification fatigue</h3>
</div>
<div class="issue-solutions">
<ul>
<li>Consolidate alerts into summaries</li>
<li>Use escalating notification levels</li>
<li>Smart filtering based on mode</li>
<li>Critical-only during certain hours</li>
</ul>
</div>
</div>

<div class="issue-card">
<div class="issue-header">
<h3>Battery-powered sensors</h3>
</div>
<div class="issue-solutions">
<ul>
<li>Regular battery check automations</li>
<li>Low battery notifications</li>
<li>Use devices with long battery life</li>
<li>Consider wired alternatives for critical sensors</li>
</ul>
</div>
