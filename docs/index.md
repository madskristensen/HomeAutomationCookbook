# The automated home

> A collection of recipes, best practices, and resources for smart home automation.

For beginners and advanced home automators alike, and for all smart home platforms.

Works with **Alexa, Siri, Google Assistant, SmartThings, Hubitat, HomeSeer, Home Assistant, Vera, Phillips Hue, Sonos**, and many others.

## Introduction

Home automation should improve people's lives by making everyday tasks effortless and intuitive. This guide provides practical, tested automation recipes that work across all major smart home platforms.

Whether you're just starting out or looking to refine your existing setup, you'll find actionable advice and real-world examples that you can implement today.
